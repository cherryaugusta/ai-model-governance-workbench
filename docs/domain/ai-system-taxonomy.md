# AI System Taxonomy

The platform manages AI-enabled systems as governed operational units.

## System examples

Seeded examples include:
- Consumer Support Triage Assistant
- Policy Summary Assistant
- Support Note Classifier
- Evidence Extractor

## Why systems matter

A system is the anchor for:
- ownership
- risk tier
- prompt versions
- model configs
- release candidates
- incidents
- audit history

## System-level metadata

Each system captures:
- name
- slug
- description
- owner team
- technical owner
- business owner
- risk tier
- domain area
- lifecycle status

## Portfolio significance

System-level governance helps the project read as an internal platform managing multiple AI-enabled workflows rather than a single-purpose app.
